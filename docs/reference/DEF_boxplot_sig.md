# Boxplot with significance stars

Variant of DEF_boxplot showing significance stars instead of p-values

## Usage

``` r
DEF_boxplot_sig(data, DE, feature, grouping)
```

## Arguments

- data:

  Data frame

- DE:

  Output from comp_means

- feature:

  Feature name

- grouping:

  Grouping variable

## Value

ggplot object
