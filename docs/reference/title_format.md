# Format plot titles for publication

Format plot titles for publication

## Usage

``` r
title_format(face = "bold", hjust = 0.5, size = 11)
```

## Arguments

- face:

  Font face (default: "bold")

- hjust:

  Horizontal justification (default: 0.5)

- size:

  Font size (default: 11)

## Value

ggplot2 theme layer
