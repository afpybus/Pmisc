# Create upset plot from binary data

Create upset plot from binary data

## Usage

``` r
upset_plot(df, value = TRUE, cex.text = 1)
```

## Arguments

- df:

  Data frame with binary/logical columns

- value:

  Value to consider as TRUE (default: TRUE)

- cex.text:

  Text size multiplier

## Value

ggplot upset plot
