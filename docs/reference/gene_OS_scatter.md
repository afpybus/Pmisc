# Gene expression vs overall survival scatter plot

Gene expression vs overall survival scatter plot

## Usage

``` r
gene_OS_scatter(df, gene)
```

## Arguments

- df:

  Data frame with time, status, and gene columns

- gene:

  Gene/feature name

## Value

ggplot object
