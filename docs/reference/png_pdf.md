# Save plot as PNG and PDF

Save plot as PNG and PDF

## Usage

``` r
png_pdf(p, filename, width, height, res = 1000)
```

## Arguments

- p:

  ggplot object

- filename:

  Filename without extension

- width:

  Width in inches

- height:

  Height in inches

- res:

  Resolution for PNG (default: 1000)

## Value

List (side effect: saves files)
