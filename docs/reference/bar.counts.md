# Create horizontal bar chart with count labels

Create horizontal bar chart with count labels

## Usage

``` r
bar.counts(df, x, fill = x, count.display = "mid")
```

## Arguments

- df:

  Data frame

- x:

  Variable name for x-axis

- fill:

  Variable name for fill (default: same as x)

- count.display:

  Label position: "mid" or "top"

## Value

ggplot object
