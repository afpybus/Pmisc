# Readjust p-values from comp_means output

Readjust p-values from comp_means output

## Usage

``` r
p.readjust(cm_out, p.adjust_method = "fdr")
```

## Arguments

- cm_out:

  Output from comp_means

- p.adjust_method:

  Adjustment method (default: "fdr")

## Value

Data frame with updated p.adj columns
